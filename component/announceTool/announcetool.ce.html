FAQ's

  • How do I update the message? - The least you need is any common text. The most you'll need is knowledge of HTML. However HTML is not necessary to know. To update the title you'll just need to change the value of the msgTitle attribute in the opening tag. To update the message contents you'll just need to enter text between the opening and closing tags.
  • I like the tool, but I'd like to change how it looks. How do I do that? - If you're confident with CSS you can simply add any additional styles to your pre-existing code. However, if you're unfamiliar or just having troubles with it than feel free to reach out to us! This is a tool that is free to use and meant to help out businesses during these times. We're more than willing to update our styles within reason.
  • I've added the code and followed the directions to the letter and it doesn't look right on my page. What did I do wrong? You may have done nothing wrong! Every site is unique, and therefore it's hard to predict every deviation that would need to be accounted for. Simply put, if you need help feel free to reach out using the contact information below and we'll update the code.
